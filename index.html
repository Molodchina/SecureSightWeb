О нас

SecureSight – это комплексное решение по безопасности ритейлеров, торговых центров, музеев и других мест массового посещения.

Предприятия получат возможность вычислять нежелательных покупателей, вооруженных людей, чрезвычайные ситуации, а также отслеживать передвижение посетителей по выбранной локации посредством видеоанализа.

Компоненты системы

SecureSight состоит из трех основных компонентов:

  • Telegram-бот: для удобного взаимодействия пользователя с системой и нейронными сетями.
  • Нейронные сети: для распознавания лиц, обнаружения оружия и выявления нестандартных ситуаций.
  • Удаленный сервер: на который поступает видеоряд с камер и на котором размещено программное обеспечение для первых двух компонентов.

Ключевые технологии

  • Компьютерное зрение: для обработки видеопотоков и распознавания объектов.
  • Машинное обучение: для обучения нейронных сетей и улучшения их точности.
  • Облачные технологии: для хранения данных и обеспечения масштабируемости.
  • API Telegram: для интеграции с Telegram и автоматизации уведомлений.

Возможности системы

  • Распознавание лиц: идентификация покупателей на основе видеопотока, включая их возраст и эмоции.
  • Обнаружение оружия: определение наличия огнестрельного оружия в общественных местах.
  • Анализ поведения: выявление аномалий в поведении людей (например, драки или паника).
  • Анализ передвижения: определение передвижения посетителей, создание тепловых карт и определение местоположения в реальном времени.
  • Мониторинг в реальном времени: поставляет оповещения о безопасности мгновенно.
